# 水土保持规范与模板资料库

本文件夹用于放置长期复用的水土保持方案编制规范、标准、地方文件、报告模板和优秀样例。

## 放置内容

- 国家及行业规范：如 GB 50433、GB/T 50434、SL 773 等。
- 法律法规及规范性文件：水土保持法、地方条例、补偿费文件、审批管理文件等。
- 报告模板：报告书、报告表、承诺书、附表、投资概算表、常用章节模板等。
- 优秀样例：已通过审查的同类项目报告、专家意见回复、典型附图清单等。

## OCR 要求

很多规范和模板 PDF 可能是扫描件，不能直接复制文字。此类文件在生成报告前应先进行文字识别。

推荐做法：

- 优先放入已经 OCR 过、可复制文字的 PDF。
- 如果只有扫描版 PDF，也可以先放入本文件夹；`context_builder.py` 会把无法抽取文字或文字量过少的 PDF 标记为“需要 OCR”。
- 对被标记为“需要 OCR”的文件，应先使用 OCR 工具生成可检索文本版 PDF、TXT 或 DOCX，再重新运行上下文构建。

## 使用规则

- 本文件夹保存长期规范和模板，不放具体项目的临时资料。
- 具体项目资料仍放项目根目录，或后续项目化目录中。
- 不要把自动生成的 `PROJECT_CONTEXT.md`、`temp_report.json`、`output_report.docx` 放入本文件夹。
